# nyaatorrent
A terminal torrent file downloader for nyaa.si (not a torrent client).

Dependencies
-----
wget

How to use
-----
Clone the repo, run main.py and follow the instructions.
The downloads are stored here: ~/Downloads/nyaatorrents/
